Hugo Setup

Jan 14, 2022
2 min read
hugo

Step 1: 安裝 Hugo

透過 Chocolaty 安裝 (Windows)

如果你正在使用 Windows 裝置 和使用 Chocolatey 作為 package management, 你可以輸入以下指令安裝 Hugo:

choco install hugo -confirm

Or if you need the “extended” Sass/SCSS version:

choco install hugo-extended -confirm

輸入以下指令確認你的安裝

hugo version



Step 2: 建立一個新的網站

hugo new site your_site_name

以上指令會為你建立一個新的 Hugo 網站 名為 your_site_name


Step 3: 加入 Theme

瀏覽 (https://themes.gohugo.io/) 查看可用的 themes
首先，在 GitHub 下載你想要的 theme 之後加進你網站的 themes 目錄

cd your_site_name
git init
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke



之後，在 config.toml 加入該 theme

theme = "ananke"



Step 4: 加入一些內容

你可以手動建立內容相關的檔案 (例如在 content/<CATEGORY>/<FILE>.<FORMAT>) 並為它設定 metadata，然而，你還可以使用 new 指令做一些額外的東西 (例如加入標題和目期):

hugo new posts/my-first-post.md



現在你可以在新建立的檔案上撰寫新的文章:

---
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true
---

Drafts do not get deployed; once you finish a post, update the header of the post to say draft: false. More info here.

Step 6: Customize the Theme

網站配置

開啟 config.toml

baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

如果你在 Hugo server 正在運行的時候修改了網站的配置或者任何其他檔案時發現在瀏覽器上沒有更新，你可能需要清除瀏覽器的 cache。



Step 7: 建置靜態網頁

很簡單，只需要輸入:

hugo -D

預設會在 ./public 輸出 (使用 -d/–destination flag 更改, 或者在 config 檔案設定 publishdir)
