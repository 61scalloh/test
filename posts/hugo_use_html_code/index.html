Hugo 使用 HTML 代碼

Jan 14, 2022
1 min read
hugo

自 0.6 版本之後, Hugo 使用了 Goldmark 作為 markdown。
因為安全起見，Goldmark 將不能使用 HTML code。

然而，如果你經常在你的網站使用 HTML，你可以加以下指令到你的 config.toml

[markup.goldmark.renderer]
  unsafe = true # Allow HTML in md files
