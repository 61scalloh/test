Laravel Setup on Windows

# 安裝 Laravel

安裝 Composer

下載 (https://getcomposer.org/download/)
 

透過 Composer 安裝 Laravel

選擇安裝位置

安裝 Laravel

 

Local Development Server

如果你已經安裝了 PHP 而你想用 PHP 內建的 development server，你可以使用 serve 指令。
這個指令會在 http://localhost:8000 啟動一個 development server :

 

# Configuration

Public Directory

安裝 Laravel 後，你需要更改你網站的根目錄 (root) 為 public 目錄。在該目錄的 index.php 會為所有進入的 HTTP requests 作為前置控制。

Configuration Files

所有 Laravel framework 的 configuration files 位於 config 目錄。

Directory Permissions

安裝 Laravel 後，你可能需要為你網站配置一些目錄的權限，在目錄 storage 和 bootstrap/cache 需要賦予 可寫入(writable) 的權限。
